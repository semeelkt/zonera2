🔓 Development Rules (Allow All Reads)

Use this during development - anyone can read data:

rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    // Allow everyone to read
    match /{document=**} {
      allow read: if true;
      allow write: if false;
    }
  }
}

✍️ Development with Write Access

If you want to test writes from your app:

rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    // Allow everyone to read and write (DEVELOPMENT ONLY!)
    match /{document=**} {
      allow read, write: if true;
    }
  }
}

⚠️ Production Rules (More Secure)

For production - authenticated users can write:

rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    // Everyone can read
    match /{document=**} {
      allow read: if true;
    }
    
    // Only authenticated users can write
    match /leagues/{leagueId} {
      allow write: if request.auth != null;
    }
    
    match /matches/{matchId} {
      allow write: if request.auth != null;
    }
  }
}
